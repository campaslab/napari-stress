2.5.2. Spherical harmonics expansion #

As explained in this notebook, a spherical harmonics expansion can be done as follows. Note that this returns a LayerDataTuple with three entries (data, properties, layer_type).

max_degree = 5

fitted_points = napari_stress.fit_spherical_harmonics(pointcloud[0], max_degree=max_degree)

fitted_points_layer = viewer.add_points(fitted_points[0], **fitted_points[1])
napari.utils.nbscreenshot(viewer, canvas_only=True)

2.5.3. Measuring curvature #

Another parameter you can retrieve from the obtained expansion with napari-stress, is the mean curvature at every point on the surface (see also glossary). Doing this requires two steps: A lebedev quadrature of the surface expansion and the measurement of curvature at the obtained quadrature points. This requires the previously obtained coefficients:

coefficients = fitted_points[1]['metadata']['spherical_harmonics_coefficients']
quadrature_points, lebedev_fit = napari_stress.lebedev_quadrature(coefficients=coefficients, number_of_quadrature_points=500, use_minimal_point_set=False)
manifold = napari_stress.create_manifold(quadrature_points, lebedev_fit=lebedev_fit, max_degree=max_degree)

Lastly, you can use napari_stress.measurements.calculate_mean_curvature_on_manifold() to calculate mean curvature. This function returns two dictionaries (features and metadata) which contain the mean curvature \(H_i\) at each point as well as the globally averaged curvature \(H_{0, arithmetic~avg}\) and \(H_{0, surf~int}\):

mean_curvature, H0_arithmetic, H_surface = measurements.calculate_mean_curvature_on_manifold(manifold)

fig, ax = plt.subplots()
hist = ax.hist(mean_curvature, bins=50)

ax.set_xlabel('Curvature')
ax.set_ylabel('Occurrences [#]')

fitted_points_layer.visible = False

2.5.4. Gauss-Bonnet test #

The Gauss-Bonnet theorem allows to evaluate the quality of the obtained surface expansion quantitatively, for details see glossary.

absolute_error, relative_error = measurements.gauss_bonnet_test(manifold)
print('Absolute error: ', absolute_error)
print('Relative error: ', relative_error)

Absolute error:  6.744764746713372e-09
Relative error:  5.367313247156943e-10

To differentiate between a “good” and a “bad” surface expansion, the stress paper suggests a threshold value of \(1\cdot10^{-2}\) for the relative error:

relative_error < 1.e-2

True

2.5.5. Ellipsoid deviation analysis #

You can also analyze, how much each of the degrees and modes of a spherical harmonics expansion contributes to the non-ellipsoidal modes of the droplet. This can be done using the deviation_from_ellipsoidal_mode() function. As usual, you find the results stored in features and metadata:

result = measurements.deviation_from_ellipsoidal_mode(pointcloud[0], max_degree=50)
signed_errors = result[1]['features']['signed_elipsoid_deviation']
coefficient_matrix = result[1]['metadata']['Elipsoid_deviation_contribution_matrix']

mpl.style.use('default')
fig, axes = plt.subplots(ncols=2, figsize=(10,5))
cmap = axes[0].imshow(np.triu(coefficient_matrix), cmap='inferno')
axes[1].imshow(np.triu(coefficient_matrix)[:10, :10], cmap='inferno')
